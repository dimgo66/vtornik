## 🚀 Быстрый старт

```bash
# 1. Клонировать
git clone <repository-url>
cd vtornik-vue

# 2. Установить зависимости
npm install

# 3. Запустить Docker
docker-compose up -d

# 4. Запустить dev-сервер
npm run dev

# 5. Открыть импорт
open scripts/import.html

# 6. Войти в CMS
# http://localhost:3003/cms/login
# admin@vtornik.online / admin123
```
