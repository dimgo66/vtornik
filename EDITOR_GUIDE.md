# Страницы редактирования статей и авторов

## 📍 Маршруты

```
/cms/article/:id/edit   → Редактирование статьи
/cms/author/:id/edit    → Редактирование автора
```

## 🎯 Доступ

**Из списка статей (CMS → Статьи):**
1. Откройте `/cms/articles`
2. Найдите нужную статью в таблице
3. Нажмите кнопку ✏️ в колонке "Действия"

**Из списка авторов (CMS → Авторы):**
1. Откройте `/cms/authors`
2. Найдите нужного автора в таблице
3. Нажмите кнопку ✏️ в колонке "Действия"

## ✏️ Редактор статей

### Панель инструментов

**Верхняя панель:**
- ← Назад — возврат к списку статей
- 👁️ Предпросмотр — просмотр статьи (в разработке)
- 💾 Сохранить — сохранение изменений

### Вкладки редактора

**📋 Основное:**
- Заголовок *
- Подзаголовок

**📖 Контент (только для статей):**

Панель инструментов с группами:

1. **Отменить/Повторить:**
   - ↩️ Undo (Ctrl+Z)
   - ↪️ Redo (Ctrl+Y)

2. **Шрифт:**
   - Выбор шрифта (PT Serif, Georgia, Arial...)
   - Выбор размера (8pt — 36pt)

3. **Форматирование текста:**
   - **B** Жирный (Ctrl+B)
   - *I* Курсив (Ctrl+I)
   - <u>U</u> Подчёркнутый (Ctrl+U)
   - ~~S~~ Зачёркнутый

4. **Цвет:**
   - 🎨 Цвет текста
   - 🖍️ Цвет фона/выделения

5. **Заголовки:**
   - H1, H2, H3
   - ¶ Параграф

6. **Списки:**
   - • Маркированный
   - 1. Нумерованный
   - → Увеличить отступ
   - ← Уменьшить отступ

7. **Выравнивание:**
   - По левому краю
   - По центру
   - По правому краю
   - По ширине

8. **Блоки:**
   - ❝ Цитата
   - `</>` Код
   - ━ Горизонтальная линия

9. **Вставка:**
   - 🖼️ Изображение
   - 🔗 Ссылка (Ctrl+K)
   - 🔓 Удалить ссылку
   - 📊 Таблица

10. **Изображения:**
    - ← Изображение слева
    - ⬌ По центру
    - → Справа
    - S/M/L/XL — размеры

11. **Стили изображений:**
    - ⬜ Скруглённые углы
    - ⬛ Рамка
    - 🌑 Тень
    - 📝 Подпись

12. **Специальное:**
    - Ω Спецсимволы
    - 🧽 Очистить форматирование
    - 🖨️ Печать

**⚙️ Параметры (боковая панель):**
- Автор *
- Номер *
- Раздел *
- Рубрика
- Жанр
- Свой жанр

**📊 Статистика:**
- Слов
- Символов
- Полос (~220 слов = 1 полоса)

### 🖼️ Работа с изображениями

**Вставка изображения:**

1. Нажмите кнопку 🖼️ на панели
2. Выберите вкладку:
   - **URL** — вставьте ссылку на изображение
   - **Загрузить** — перетащите файл или кликните для выбора
3. Настройте параметры:
   - Выравнивание (слева/центр/справа)
   - Размер (S/M/L/XL)
   - Подпись
   - Стили (скруглённые углы, рамка, тень)
4. Нажмите "Вставить"

**Редактирование существующего:**

1. Кликните на изображение в редакторе
2. Используйте кнопки на панели:
   - ← ⬌ → — выравнивание
   - S M L XL — размер
   - ⬜ ⬛ 🌑 📝 — стили

### ⌨️ Горячие клавиши

- **Ctrl+B** — жирный
- **Ctrl+I** — курсив
- **Ctrl+K** — вставить ссылку
- **Ctrl+Z** — отменить
- **Ctrl+Y** — повторить

## ✏️ Редактирование автора

**📋 Основное:**
- Имя *
- Статус / Подпись
- Биография
- Фото автора (URL)

**⚙️ Параметры:**
- Фотография (предпросмотр + URL)

**📊 Статистика:**
- Количество публикаций

## 💾 Сохранение

**Кнопка "Сохранить":**
1. Проверяет обязательные поля (*)
2. Отправляет данные в базу (Supabase или localStorage)
3. Возвращает к списку
4. Показывает уведомление об успехе/ошибке

**Автосохранение:**
- В разработке
- Планируется каждые 30 секунд

## 📤 Загрузка изображений

**В базу данных:**
```javascript
// Сервис imageUpload.js
const result = await uploadImage(file)
// result = { id, url, filename, size, type }
```

**Хранение:**
- **Локально:** localStorage (демо режим)
- **Облако:** Supabase Storage (продакшен)
- **Локально:** Docker Supabase (разработка)

## 🔄 Повторное использование

**Все изображения сохраняются в базе:**

1. Откройте модальное окно изображений
2. Переключитесь на вкладку "Галерея" (в разработке)
3. Выберите ранее загруженное изображение
4. Вставьте в статью

**Все статьи и авторы доступны для выбора:**
- При создании новой статьи
- При редактировании номера
- При просмотре статистики

## 🐛 Решение проблем

**Изображение не вставляется:**
- Проверьте формат файла (PNG, JPG, GIF)
- Проверьте размер (максимум 5MB)
- Попробуйте вставить по URL

**Текст не форматируется:**
- Выделите текст перед применением формата
- Проверьте, что редактор активен (фокус)

**Сохранение не работает:**
- Проверьте подключение к базе данных
- Проверьте заполнение обязательных полей (*)
- Откройте консоль браузера (F12) для ошибок
