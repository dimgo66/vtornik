# Project Brief: ВТОРНИК

## 📖 Описание проекта

**ВТОРНИК** — литературно-художественный журнал, веб-приложение на Vue 3 + Vite + Pinia.

## 🎯 Цели

1. Публикация литературных произведений (проза, поэзия, эссе)
2. Представление авторов и их работ
3. CMS для управления контентом (номера, статьи, авторы)
4. Генерация PDF для печати

## 📦 Объем проекта

### Публичная часть
- Главная страница с номерами
- Страница номера (оглавление)
- Страница статьи (полный текст)
- Страницы авторов (список + детально)
- Страница «О журнале»

### CMS (админ-панель)
- Управление номерами журнала
- Управление статьями (редактор Rich Text)
- Управление авторами
- Подготовка к печати
- Калькулятор корешка

## 🎨 Дизайн

Нейроморфный дизайн с тёплой цветовой палитрой:
- `--cr: #7A1515` — красный акцент
- `--paper: #F9F5F0` — фон бумаги
- `--ink: #1C1410` — текст
- `--gold: #C9A962` — золото

## 📊 Данные

**Текущее состояние:** JSON файлы в `src/data/`
**Цель:** Интеграция с Supabase (локально + облако)
