# Product Context

## Зачем существует этот проект

ВТОРНИК — цифровая платформа для публикации и распространения литературно-художественных произведений. Проект позволяет:

1. **Авторам** — публиковать свои работы, иметь персональную страницу
2. **Редакции** — управлять контентом через удобную CMS
3. **Читателям** — удобно читать произведения, знакомиться с авторами

## Какие проблемы решает

| Проблема | Решение |
|----------|---------|
| Сложность публикации | Простая CMS с Rich Text редактором |
| Отсутствие централизованной платформы | Единый сайт с номерами и авторами |
| Печать физических копий | Генерация PDF для типографии |
| Управление архивом | База данных всех публикаций |

## Как он должен работать

### Пользовательский сценарий (читатель)
1. Открывает главную страницу
2. Выбирает номер журнала
3. Кликает на статью в оглавлении
4. Читает произведение
5. Может перейти к автору

### Пользовательский сценарий (редактор)
1. Авторизуется в CMS
2. Создаёт/редактирует номер
3. Добавляет статьи через Rich Text редактор
4. Загружает изображения
5. Генерирует PDF для печати

## Цели пользовательского опыта

- **Минимализм** — ничего лишнего, фокус на контенте
- **Тёплая атмосфера** — бумажная текстура, спокойные цвета
- **Быстрая загрузка** — оптимизированные изображения, code splitting
- **Доступность** — семантическая вёрстка, контрастность
