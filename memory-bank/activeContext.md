## Активные решения и соображения

### Supabase интеграция
- Использовать официальный клиент `@supabase/supabase-js`
- Создать единый сервис `src/services/api.js`
- Обновить все stores для работы с API

### Аутентификация
- Email/password через Supabase Auth
- Token в localStorage
- Router guard для защиты `/cms/*`

### Rich Text Editor
- Текущая реализация через `document.execCommand` (deprecated, но работает)
- Рассмотреть переход на TipTap или Quill в будущем
